# vue_accounts_exercise

## Краткое описание

Небольшое веб-приожение, симулирующее управление учётными записями пользователей. При перезагрузке страницы валидные записи сохраняются.

## Установка зависимостей

`npm i`

## Запуск проекта в режиме разработки

`npm run dev`

## Сборка и запуск проекта в продуктовом режиме

_При необходимости установите утилиту serve `npm install -g serve`_

`npm run build && serve -s dist`

## Зависимости

- ts
- vue (фронтенд фреймворк)
- pinia (стэйт менеджер)
- tailwind (CSS фреймфорк)
- eslint (линтер)
- oxlint (линтер)

_Для получения более подробной информации ознакомьтесь с файлом `package.json`_
